Programming

On this module students learn about algorithms, how they are defined and how they are implemented in code, this is undertaken in the following stages:

  1. Students undertake study of the features of an algorithm and define the written rules used in it's implementation through producing a technical report and powerpoint presentation.
  2. Students then study the heap sort algorithm and implement the algorithm in an IDE.
  3. Students are then asked to create a program and work closely with staff to create a database.
  4. Students can use time on the module to write their own programs and implement their own algorithms using the Little Book of Programming Challenges as displayed in figure 1.
